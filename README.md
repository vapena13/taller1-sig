# Taller 1 SIG – Benchmarking Sentinel-2

Repositorio con la implementación del benchmark comparando R (terra, stars), Python (rasterio) y Julia (Rasters.jl) para procesamiento raster de Sentinel-2.

Estructura:
- notebooks: ejecución en JupyterLab
- scripts: ejecución desde terminal
- data: rutas compartidas al raster
- images: figuras generadas
- respuestas.qmd: documento de análisis

Asignatura: Programación SIG  
Maestría en Geomática
